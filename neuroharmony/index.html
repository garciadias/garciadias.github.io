NEUROHARMONY - Dataset statistical description

The links below provide a comprehensive description of the datasets used in the development of the NEUROHARMONY tool, presented in the article "Neuroharmony: A new approach for harmonizing volumetric MRI data for unseen scanners". The pages were generated using the Pandas Profiling library. The details in the Pandas Profiling configurations are given in the "reproduction" tab, which can be found within the description of each page. More information about the structure of the pages can be found in the Pandas Profiling documentation. The description of the variables is presented in the appendix of the article. If you have any questions, please contact rafael.garcia_dias@kcl.ac.uk.
